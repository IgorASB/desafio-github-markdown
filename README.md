# 📝 Desafio GitHub Markdown

Repositório criado para praticar e demonstrar o uso de **Markdown no GitHub**, explorando a formatação de texto, estrutura de documentos e boas práticas de documentação em projetos de software.

---

## 📌 Sobre o Projeto

Este repositório faz parte de um desafio prático de aprendizado com foco em:

- Sintaxe básica e avançada do Markdown
- Criação de READMEs informativos e bem estruturados
- Uso de recursos do GitHub como badges, emojis e tabelas
- Boas práticas de documentação em projetos open source

---

## 🚀 Conceitos Praticados

| Recurso           | Descrição                                              |
|-------------------|--------------------------------------------------------|
| Títulos           | Hierarquia com `#`, `##`, `###`                        |
| Listas            | Ordenadas e não ordenadas                              |
| Código            | Inline e blocos com syntax highlight                  |
| Tabelas           | Organização de dados em formato tabular                |
| Links e Imagens   | Inserção de URLs e imagens                             |
| Ênfase            | **Negrito**, *itálico*, ~~tachado~~                    |
| Checkboxes        | Listas de tarefas com `- [ ]`                          |
| Citações          | Blocos de citação com `>`                              |
